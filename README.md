## 📌 教學重點摘要

1. **基礎設置** — 使用權限說明、建立 Figma Make 檔案
2. **AI 生成** — 如何撰寫 AI 指令來生成 UI 介面
3. **精細調整** — Point and Edit 互動式編輯功能
4. **進階整合** — 匯入設計系統、圖片附件、Figma 設計檔
5. **後端連接** — 接上 Supabase 建立真實後台資料
6. **發佈上線** — Publish 功能將設計部署為網站
