## 📋 各階段重點

### 第一階段：Figma AI 基本使用
- 認識 Figma Make 介面與使用權限
- 建立第一個 Make 檔案
- 學會基本 AI 指令（Prompt）寫法

### 第二階段：Icon 建置與修改
- 用 AI 指令生成 icon 組合
- 使用 Point and Edit 修改單一 icon
- 統一 icon 風格（大小、線條、顏色）

### 第三階段：畫面建置與修改
- 用 AI 生成完整 App 畫面
- 套用設計系統與 icon
- Point and Edit 細部調整，Publish 發佈
