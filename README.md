# Figma AI Demo 教學專案

> Figma Make AI UI/UX 設計教學，共四個階段，每階段約 5 分鐘。

---

## 🗂️ 教學四階段（共約 20 分鐘）

| 階段 | 主題 | 時間 | 資料夾 |
|------|------|------|--------|
| 第一階段 | Figma AI 基本使用 | ~5 分鐘 | [`01-figma-ai-basics`](./01-figma-ai-basics/) |
| 第二階段 | Icon 建置與修改 | ~5 分鐘 | [`02-icon-design`](./02-icon-design/) |
| 第三階段 | 畫面建置與修改 | ~5 分鐘 | [`03-screen-design`](./03-screen-design/) |
| 第四階段 | 建置免費後台串接 | ~5 分鐘 | [`04-backend-integration`](./04-backend-integration/) |

---

## 📋 各階段重點

### 第一階段：Figma AI 基本使用
- 認識 Figma Make 介面與使用權限
- 建立第一個 Make 檔案
- 學會基本 AI 指令（Prompt）寫法

### 第二階段：Icon 建置與修改
- 用 AI 指令生成 icon 組合
- 使用 Point and Edit 修改單一 icon
- 統一 icon 風格（大小、線條、顏色）

### 第三階段：畫面建置與修改
- 用 AI 生成完整 App 畫面
- 套用設計系統與 icon
- Point and Edit 細部調整，Publish 發佈

### 第四階段：建置免費後台串接
- 介紹可免費使用的後台服務（Supabase、Firebase、Airtable）
- 建立 Supabase 資料庫與資料表
- 在 Figma Make 中串接真實後台資料
