# 第一階段：Figma AI 基本使用（約 5 分鐘）

## 🎯 學習目標
- 了解 Figma Make 是什麼
- 知道如何開啟與建立 Figma Make 檔案
- 學會基本 AI 指令的寫法

## 📝 教學步驟

1. **介紹 Figma Make**
   - 前往 https://www.figma.com/make/
   - 說明使用權限（免費/付費方案差異）

2. **建立第一個 Figma Make 檔案**
   - 點選「New Make file」
   - 認識介面佈局（Prompt 區、預覽區、圖層區）

3. **第一個 AI 指令**
   - 示範基本 Prompt 寫法
   - 範例：`設計一個簡單的登入頁面，包含 Logo、帳號密碼輸入框、登入按鈕`

## 💡 Prompt 寫法技巧
- 說明**功能**：這個頁面要做什麼
- 說明**風格**：簡約、圓角、配色
- 說明**元素**：需要哪些 UI 元件
