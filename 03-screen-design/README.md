# 第三階段：畫面建置與修改（約 5 分鐘）

## 🎯 學習目標
- 用 AI 生成完整 App 畫面
- 套用設計系統（Design System）
- 發佈為可預覽的網頁

## 📝 教學步驟

1. **生成完整畫面**
   - 範例 Prompt：
     ```
     設計一個醫療預約 App 的首頁
     包含：頂部問候語、今日預約卡片、科別快捷 icon、最近紀錄列表
     風格：白底、藍色主色調、圓角卡片、現代簡約
     ```

2. **套入設計系統**
   - 使用「加入設計系統」功能
   - 將第二階段製作的 icon 套用進畫面
   - 統一字體、顏色、間距

3. **加入附件素材**
   - 匯入圖片（大頭照、插圖）
   - 匯入 Figma 設計檔元件

4. **Point and Edit 細部調整**
   - 點選卡片 → 調整圓角、陰影
   - 點選文字 → 修改字體大小
   - 點選按鈕 → 更換顏色風格

5. **Publish 發佈**
   - 點選右上角「Publish」
   - 取得可分享的網址
   - 示範在手機瀏覽器開啟預覽

## 💡 畫面設計技巧
- 先生成**骨架**，再用 Point and Edit **細修**
- 每次修改範圍不要太大，分段調整效果更好
- 善用「加入 Figma 設計」功能套用既有元件庫
