# 第二階段：Icon 建置與修改（約 5 分鐘）

## 🎯 學習目標
- 用 AI 指令生成 icon 組合
- 使用 Point and Edit 修改單一 icon
- 統一 icon 風格（大小、線條、顏色）

## 📝 教學步驟

1. **用 AI 指令生成 Icon 組**
   - 範例 Prompt：
     ```
     幫我建立一組導覽列 icon，包含：首頁、搜尋、收藏、個人
     風格：線條式（outline），24x24px，圓角，顏色 #333333
     ```

2. **Point and Edit 修改 Icon**
   - 點選畫面上的 icon
   - 在 Prompt 輸入修改指令，例如：
     ```
     將這個 icon 改成填滿式（filled）風格
     ```
     ```
     將 icon 顏色改為藍色 #3B82F6
     ```

3. **批次修改風格**
   - 一次調整所有 icon 的一致性
   - 範例：`統一所有 icon 的大小為 24px，線條粗細 1.5px`

## 💡 Icon 設計技巧
- **Outline（線條式）**：適合導覽列未選取狀態
- **Filled（填滿式）**：適合導覽列選取狀態
- 建議尺寸：16px / 20px / 24px / 32px
